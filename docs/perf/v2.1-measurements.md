## How to re-capture

If these numbers need to be re-measured against a future build, the cheapest path is:

1. `xcodebuild -project MDViewer.xcodeproj -scheme MDViewer -configuration Release -derivedDataPath build clean build`
2. Open Instruments (Xcode тЖТ Open Developer Tool тЖТ Instruments).
3. For PERF-04..06: pick the **Logging** template, target the built `.app`, set launch arg to one of the test files, hit Record.
4. For PERF-07: pick the **Allocations** template, target the built `.app`, repeatedly open `large-100kb.md`.
5. Filter os_signpost track to subsystem `com.mdviewer.app`, category `RenderingPipeline`. Read `open-to-paint` interval duration.
6. Update this file in place with new numbers + new commit SHA.
